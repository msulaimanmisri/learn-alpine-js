Directive Name : Cloak

Link : Alpine.js Documentation

Kadang2 kita akan nampak macam ada glitch dalam paparan front-end kita. Ada orang panggil `Content Jumping`, ada orang panggil dia sebagai `Content Cloak`. Biasanya terjadi bila kita nak hide sesuatu Node dengan menggunakan JavaScipt.

Cara untuk elak perkara ini berlaku adalah dengan menggunakan x-cloak. Masukkan code ini pada header file anda

[x-cloak] {
display: none !important;
}
