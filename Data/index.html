Link : Alpine.js Documentation

x-data merupakan initialize Directive Syntax dalam Alpine.js. Setiap code kita perlu dimulakan dengannya. Dan x-data sepatutnya diletakkan pada parent Node. Tetapi ianya juga boleh selari dengan Node yang kita nak pakai denngan Directive lain.
